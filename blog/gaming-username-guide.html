üì± Platform-Specific Tips

Xbox Gamertag Tips

  • Maximum 12 characters (Microsoft added suffix numbers if taken)
  • One free name change, then $10 each
  • Avoid offensive terms‚ÄîXbox enforces strict policies
  • Numbers are automatically added if exact name is taken

PlayStation Name Tips

  • 3-16 characters allowed
  • First change is free, then $5-$10
  • Some older games may have issues with changed names
  • Can include letters, numbers, hyphens, and underscores

Steam Username Tips

  • No character limit for display name
  • Change anytime for free
  • Login name cannot be changed (only display name)
  • Supports special characters and unicode

Discord Username Tips

  • 2-32 characters for display name
  • Unique handles now required (like @username)
  • Case insensitive for handles
  • Frequently change display name without affecting handle
